获取版本间差异文件上线

开发环境->内网测试环境->外网UAT环境->正式环境

0、进入到项目目录

[root@localhost /]# cd /data/xiaochengxu/
[root@localhost xiaochengxu]# pwd
/data/xiaochengxu

[root@localhost xiaochengxu]# git log --pretty=oneline
